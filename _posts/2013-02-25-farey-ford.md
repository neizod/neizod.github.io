**‡∏ô‡∏¥‡∏¢‡∏≤‡∏° 1** ‡∏•‡∏≥‡∏î‡∏±‡∏ö Farey ‡∏≠‡∏±‡∏ô‡∏î‡∏±‡∏ö $n$ ‡πÄ‡∏£‡∏µ‡∏¢‡∏Å‡∏¢‡πà‡∏≠‡∏ß‡πà‡∏≤ $F_n$ ‡∏Ñ‡∏∑‡∏≠‡∏•‡∏≥‡∏î‡∏±‡∏ö‡∏Ç‡∏≠‡∏á‡πÄ‡∏®‡∏©‡∏™‡πà‡∏ß‡∏ô‡∏≠‡∏¢‡πà‡∏≤‡∏á‡∏ï‡πà‡∏≥‡πÉ‡∏ô‡∏ä‡πà‡∏ß‡∏á $[0, 1]$
‡∏ã‡∏∂‡πà‡∏á‡∏ï‡∏±‡∏ß‡∏™‡πà‡∏ß‡∏ô‡∏ô‡∏±‡πâ‡∏ô‡∏°‡∏µ‡∏Ñ‡πà‡∏≤‡∏ô‡πâ‡∏≠‡∏¢‡∏Å‡∏ß‡πà‡∏≤‡πÄ‡∏ó‡πà‡∏≤‡∏Å‡∏±‡∏ö $n$ ‡πÅ‡∏•‡∏∞‡∏™‡∏°‡∏≤‡∏ä‡∏¥‡∏Å‡πÅ‡∏ï‡πà‡∏•‡∏∞‡∏ï‡∏±‡∏ß‡πÄ‡∏£‡∏µ‡∏¢‡∏á‡∏•‡∏≥‡∏î‡∏±‡∏ö‡∏à‡∏≤‡∏Å‡∏ô‡πâ‡∏≠‡∏¢‡πÑ‡∏õ‡∏°‡∏≤‡∏Å

**‡∏™‡∏°‡∏ö‡∏±‡∏ï‡∏¥ 1.1** $F_n$ ‡∏à‡∏∞‡∏ö‡∏£‡∏£‡∏à‡∏∏‡∏™‡∏°‡∏≤‡∏ä‡∏¥‡∏Å‡∏ó‡∏∏‡∏Å‡∏ï‡∏±‡∏ß‡∏à‡∏≤‡∏Å $F_k$ ‡∏™‡∏≥‡∏´‡∏£‡∏±‡∏ö‡∏ó‡∏∏‡∏Å‡πÜ $k \le n$

**‡∏™‡∏°‡∏ö‡∏±‡∏ï‡∏¥ 1.2** ‡∏™‡∏°‡∏≤‡∏ä‡∏¥‡∏Å‡∏ó‡∏µ‡πà‡πÄ‡∏û‡∏¥‡πà‡∏°‡πÄ‡∏Ç‡πâ‡∏≤‡∏°‡∏≤‡πÉ‡∏ô $F_n$ ‡∏à‡∏≤‡∏Å $F_{n‚àí1}$ ‡∏Ñ‡∏∑‡∏≠ $\lbrace a/n \mid 0 < a < n, \gcd(a, n) = 1 \rbrace$

**‡∏™‡∏°‡∏ö‡∏±‡∏ï‡∏¥ 1.3** ‡πÉ‡∏´‡πâ $\abs{F_n}$ ‡πÅ‡∏ó‡∏ô‡∏à‡∏≥‡∏ô‡∏ß‡∏ô‡∏™‡∏°‡∏≤‡∏ä‡∏¥‡∏Å‡∏Ç‡∏≠‡∏á $F_n$ ‡∏à‡∏∞‡πÑ‡∏î‡πâ‡∏ß‡πà‡∏≤ $\abs{F_n} = \abs{F_{n-1}} + \phi(n)$  

**‡∏™‡∏°‡∏ö‡∏±‡∏ï‡∏¥ 1.4** ‡∏à‡∏≤‡∏Å $\abs{F_1} = 2$ ‡∏î‡∏±‡∏á‡∏ô‡∏±‡πâ‡∏ô $\abs{F_n} = 1 + \sum\limits_{k=1}^n \phi(k)$

**‡∏™‡∏°‡∏ö‡∏±‡∏ï‡∏¥ 1.5** ‡∏™‡∏°‡∏≤‡∏ä‡∏¥‡∏Å‡∏ï‡∏±‡∏ß‡∏ó‡∏µ‡πà‡∏™‡∏≠‡∏á‡∏Ç‡∏≠‡∏á $F_n$ ‡∏°‡∏µ‡∏Ñ‡πà‡∏≤‡πÄ‡∏ó‡πà‡∏≤‡∏Å‡∏±‡∏ö $1/n$

{: .remarkbox}
> [‡∏≠‡∏≠‡∏¢‡πÄ‡∏•‡∏≠‡∏£‡πå‡∏ó‡∏≠‡πÄ‡∏ó‡∏µ‡∏¢‡∏ô‡∏ï‡πå][euler totient] (Euler's totient) ‡∏™‡∏±‡∏ç‡∏•‡∏±‡∏Å‡∏©‡∏ì‡πå $\phi(n)$ ‡∏Ñ‡∏∑‡∏≠‡∏ü‡∏±‡∏á‡∏Å‡πå‡∏ä‡∏±‡∏ô‡πÅ‡∏™‡∏î‡∏á‡∏à‡∏≥‡∏ô‡∏ß‡∏ô‡∏ô‡∏±‡∏ö‡πÉ‡∏ô‡∏ä‡πà‡∏ß‡∏á $[1, n]$ ‡∏ó‡∏µ‡πà‡πÄ‡∏õ‡πá‡∏ô‡∏à‡∏≥‡∏ô‡∏ß‡∏ô‡πÄ‡∏â‡∏û‡∏≤‡∏∞‡∏™‡∏±‡∏°‡∏û‡∏±‡∏ó‡∏ò‡πå‡∏Å‡∏±‡∏ö $n$ ‡∏°‡∏µ‡∏™‡∏π‡∏ï‡∏£‡∏ó‡∏±‡πà‡∏ß‡πÑ‡∏õ‡∏Ñ‡∏∑‡∏≠
> 
> $$
> \begin{align}
>    \phi(1) &= 1 \\
> \phi(p^kq) &= p^{k-1}(p - 1)\phi(q), \quad\quad \gcd(p^k, q) = 1
> \end{align}
> $$

**‡∏Ç‡∏±‡πâ‡∏ô‡∏ï‡∏≠‡∏ô‡∏ß‡∏¥‡∏ò‡∏µ 1** ‡∏Å‡∏≤‡∏£‡∏™‡∏£‡πâ‡∏≤‡∏á‡∏•‡∏≥‡∏î‡∏±‡∏ö Farey ‡∏î‡πâ‡∏ß‡∏¢‡∏Å‡∏≤‡∏£‡πÑ‡∏•‡πà‡∏ó‡∏∏‡∏Å‡πÄ‡∏®‡∏©‡∏™‡πà‡∏ß‡∏ô‡∏ó‡∏µ‡πà‡πÄ‡∏õ‡πá‡∏ô‡πÑ‡∏õ‡πÑ‡∏î‡πâ ‡πÅ‡∏•‡πâ‡∏ß‡πÄ‡∏Å‡πá‡∏ö‡πÄ‡∏â‡∏û‡∏≤‡∏∞‡πÄ‡∏®‡∏©‡∏™‡πà‡∏ß‡∏ô‡∏≠‡∏¢‡πà‡∏≤‡∏á‡∏ï‡πà‡∏≥

‡∏à‡∏≤‡∏Å‡∏™‡∏°‡∏ö‡∏±‡∏ï‡∏¥ 1.2 ‡∏™‡∏≤‡∏°‡∏≤‡∏£‡∏ñ‡πÄ‡∏Ç‡∏µ‡∏¢‡∏ô‡∏ü‡∏±‡∏á‡∏Å‡πå‡∏ä‡πâ‡∏ô‡πÉ‡∏ô‡∏†‡∏≤‡∏©‡∏≤ Python ‡∏™‡∏≥‡∏´‡∏£‡∏±‡∏ö‡∏´‡∏≤ $F_n$ ‡πÑ‡∏î‡πâ‡∏î‡∏±‡∏á‡∏ô‡∏µ‡πâ

``` python
def farey(n):
    sequence = [Fraction(0), Fraction(1)]
    for b in range(1, n+1):
        for a in range(1, b):
            if gcd(a, b) == 1:
                sequence += [Fraction(a, b)]
    return sorted(sequence)
```

**‡∏ô‡∏¥‡∏¢‡∏≤‡∏° 2** ‡∏Ñ‡πà‡∏≤ *mediant* ‡∏°‡∏µ‡∏Ñ‡πà‡∏≤‡πÄ‡∏ó‡πà‡∏≤‡∏Å‡∏±‡∏ö $\frac{a+c}{b+d}$ ‡∏™‡∏≥‡∏´‡∏£‡∏±‡∏ö‡πÄ‡∏®‡∏©‡∏™‡πà‡∏ß‡∏ô‡∏≠‡∏¢‡πà‡∏≤‡∏á‡∏ï‡πà‡∏≥ $\frac{a}{b}, \frac{c}{d}$

**‡∏ó‡∏§‡∏©‡∏é‡∏µ‡∏ö‡∏ó 2.1** ‡∏ñ‡πâ‡∏≤ $\frac{a}{b} < \frac{c}{d}$ ‡πÅ‡∏•‡πâ‡∏ß $\frac{a}{b} < \frac{a+c}{b+d} < \frac{c}{d}$

**‡∏û‡∏¥‡∏™‡∏π‡∏à‡∏ô‡πå** ‡∏à‡∏≤‡∏Å $\frac{a}{b} < \frac{c}{d}$ ‡∏à‡∏∞‡πÑ‡∏î‡πâ‡∏ß‡πà‡∏≤

**‡∏û‡∏¥‡∏™‡∏π‡∏à‡∏ô‡πå** ‡πÉ‡∏´‡πâ $\frac{a}{b}$ ‡πÄ‡∏õ‡πá‡∏ô‡∏™‡∏°‡∏≤‡∏ä‡∏¥‡∏Å‡∏Ç‡∏≠‡∏á $F_n$ ‡πÉ‡∏î‡πÜ

**‡∏û‡∏¥‡∏™‡∏π‡∏à‡∏ô‡πå** ‡πÄ‡∏•‡∏∑‡∏≠‡∏Å 3 ‡∏û‡∏à‡∏ô‡πå‡∏ï‡∏¥‡∏î‡∏Å‡∏±‡∏ô‡πÉ‡∏ô $F_n$ ‡∏Ñ‡∏∑‡∏≠ $\frac{a}{b}, \frac{c}{d}, \frac{e}{f}$ ‡∏à‡∏∞‡πÑ‡∏î‡πâ‡∏ß‡πà‡∏≤ $bc - ad = de - cf$ ‡πÄ‡∏ô‡∏∑‡πà‡∏≠‡∏á‡∏à‡∏≤‡∏Å‡∏™‡∏°‡∏Å‡∏≤‡∏£‡∏ô‡∏µ‡πâ‡∏à‡∏£‡∏¥‡∏á‡∏™‡∏≥‡∏´‡∏£‡∏±‡∏ö‡∏ó‡∏∏‡∏Å‡πÜ 3 ‡∏û‡∏à‡∏ô‡πå‡πÉ‡∏î‡πÜ ‡∏ó‡∏µ‡πà‡∏ï‡∏¥‡∏î‡∏Å‡∏±‡∏ô‡πÉ‡∏ô $F_n$ ‡∏î‡∏±‡∏á‡∏ô‡∏±‡πâ‡∏ô $1 = 1\cdot1 - n\cdot0 = \dots = bc - ad = de - cf = \dots$

**‡∏ó‡∏§‡∏©‡∏é‡∏µ‡∏ö‡∏ó 2.4** ‡∏™‡∏≥‡∏´‡∏£‡∏±‡∏ö $\frac{a}{b},\frac{c}{d}$ ‡∏ó‡∏µ‡πà‡∏≠‡∏¢‡∏π‡πà‡∏ï‡∏¥‡∏î‡∏Å‡∏±‡∏ô‡πÉ‡∏ô $F_n$ ‡πÅ‡∏•‡πâ‡∏ß $\gcd(a+c,b+d) = 1$

**‡∏û‡∏¥‡∏™‡∏π‡∏à‡∏ô‡πå** ‡πÄ‡∏û‡∏£‡∏≤‡∏∞ $1 = bc - ad = (a+c)b - (b+d)a$ ‡∏´‡∏£‡∏∑‡∏≠‡∏™‡∏≤‡∏°‡∏≤‡∏£‡∏ñ‡πÄ‡∏Ç‡∏µ‡∏¢‡∏ô‡πÑ‡∏î‡πâ‡∏≠‡∏µ‡∏Å‡∏≠‡∏¢‡πà‡∏≤‡∏á‡∏ß‡πà‡∏≤ $1 = (a+c)x-(b+d)z$ ‡πÇ‡∏î‡∏¢‡∏≠‡∏±‡∏ï‡∏•‡∏±‡∏Å‡∏©‡∏ì‡πå‡∏Ç‡∏≠‡∏á‡πÄ‡∏ö‡∏ã‡∏π ‡∏à‡∏∞‡πÑ‡∏î‡πâ‡∏ß‡πà‡∏≤ $\gcd(a+c,b+d)=1$

**‡∏Ç‡∏±‡πâ‡∏ô‡∏ï‡∏≠‡∏ô‡∏ß‡∏¥‡∏ò‡∏µ 2** ‡∏Å‡∏≤‡∏£‡∏™‡∏£‡πâ‡∏≤‡∏á‡∏•‡∏≥‡∏î‡∏±‡∏ö Farey ‡∏î‡πâ‡∏ß‡∏¢‡∏Å‡∏≤‡∏£‡∏™‡∏£‡πâ‡∏≤‡∏á‡∏•‡∏≥‡∏î‡∏±‡∏ö Farey (‡∏î‡πâ‡∏ß‡∏¢‡∏Å‡∏≤‡∏£‡∏™‡∏£‡πâ‡∏≤‡∏á‡∏•‡∏≥‡∏î‡∏±‡∏ö Farey (...))

‡∏≠‡∏¢‡πà‡∏≤‡∏á‡πÑ‡∏£‡∏Å‡πá‡∏ï‡∏≤‡∏° ‡∏ß‡∏¥‡∏ò‡∏µ‡∏ó‡∏µ‡πà 1 ‡πÄ‡∏õ‡πá‡∏ô‡∏Å‡∏≤‡∏£‡∏´‡∏≤‡∏™‡∏°‡∏≤‡∏ä‡∏¥‡∏Å‡∏ó‡∏∏‡∏Å‡∏ï‡∏±‡∏ß‡πÅ‡∏•‡πâ‡∏ß‡∏ô‡∏≥‡∏°‡∏≤‡πÄ‡∏£‡∏µ‡∏¢‡∏á‡∏•‡∏≥‡∏î‡∏±‡∏ö ‡πÅ‡∏ï‡πà‡∏ß‡∏¥‡∏ò‡∏µ‡∏ó‡∏µ‡πà 2 ‡∏ô‡∏±‡πâ‡∏ô ‡∏à‡∏∞‡πÄ‡∏õ‡πá‡∏ô‡∏Å‡∏≤‡∏£‡∏´‡∏≤‡∏™‡∏°‡∏≤‡∏ä‡∏¥‡∏Å‡∏Ç‡∏≠‡∏á‡∏•‡∏≥‡∏î‡∏±‡∏ö‡πÑ‡∏õ‡∏û‡∏£‡πâ‡∏≠‡∏°‡πÜ ‡∏Å‡∏±‡∏ö‡∏Å‡∏≤‡∏£‡∏Å‡∏≥‡∏´‡∏ô‡∏î‡∏ï‡∏≥‡πÅ‡∏´‡∏ô‡πà‡∏á‡πÉ‡∏ô‡∏•‡∏≥‡∏î‡∏±‡∏ö ‡∏î‡∏±‡∏á‡∏ô‡∏±‡πâ‡∏ô‡∏à‡∏∞‡∏´‡∏≤ $F_n$ ‡πÇ‡∏î‡∏¢‡∏≠‡∏≤‡∏®‡∏±‡∏¢‡∏ó‡∏§‡∏©‡∏é‡∏µ‡∏ö‡∏ó‡∏ó‡∏µ‡πà 2.2 ‡πÅ‡∏•‡∏∞ 2.4

``` python
def farey(n):
    if n == 1:
        return [Fraction(0), Fraction(1)]
    sequence = []
    for g in farey(n-1):
        if sequence:
            b = f.denominator + g.denominator
            if b == n:
                a = f.numerator + g.numerator
                sequence += [Fraction(a, b)]
        f = g
        sequence += [g]
    return sequence
```

**‡∏ó‡∏§‡∏©‡∏é‡∏µ‡∏ö‡∏ó 2.5** ‡∏™‡∏≥‡∏´‡∏£‡∏±‡∏ö $\frac{a}{b}, \frac{c}{d}$ ‡∏ó‡∏µ‡πà‡∏≠‡∏¢‡∏π‡πà‡∏ï‡∏¥‡∏î‡∏Å‡∏±‡∏ô‡πÉ‡∏ô $F_n$ ‡∏à‡∏∞‡πÑ‡∏î‡πâ‡∏ß‡πà‡∏≤‡∏û‡∏à‡∏ô‡πå‡∏ñ‡∏±‡∏î‡πÑ‡∏õ‡∏Ñ‡∏∑‡∏≠ $\frac{kc-a}{kd-b}$ ‡πÇ‡∏î‡∏¢‡∏ó‡∏µ‡πà $k = \left\lfloor \frac{n+b}{d} \right\rfloor$

**‡∏û‡∏¥‡∏™‡∏π‡∏à‡∏ô‡πå** ‡πÉ‡∏´‡πâ‡∏û‡∏à‡∏ô‡πå‡∏ñ‡∏±‡∏î‡πÑ‡∏õ‡∏ó‡∏µ‡πà‡∏ï‡πâ‡∏≠‡∏á‡∏Å‡∏≤‡∏£‡∏´‡∏≤‡∏Ñ‡∏∑‡∏≠ $\frac{e}{f}$ ‡∏à‡∏∞‡πÑ‡∏î‡πâ‡∏ß‡πà‡∏≤ $\frac{c}{d} = \frac{a+e}{b+f}$ ‡πÅ‡∏ï‡πà‡πÄ‡∏ô‡∏∑‡πà‡∏≠‡∏á‡∏à‡∏≤‡∏Å $\gcd(a+e,b+f) = k \ge 1$ ‡∏î‡∏±‡∏á‡∏ô‡∏±‡πâ‡∏ô $kc = a + e$ ‡πÅ‡∏•‡∏∞ $kd = b + f$ ‡πÇ‡∏î‡∏¢‡∏ó‡∏µ‡πà $kd - b = f \le n$ ‡πÅ‡∏•‡∏∞‡∏ó‡∏≥‡πÉ‡∏´‡πâ‡πÑ‡∏î‡πâ‡∏ß‡πà‡∏≤ $k = \left\lfloor \frac{n+b}{d} \right\rfloor \le \frac{n+b}{d}$

**‡∏Ç‡∏±‡πâ‡∏ô‡∏ï‡∏≠‡∏ô‡∏ß‡∏¥‡∏ò‡∏µ 3** ‡∏Å‡∏≤‡∏£‡∏™‡∏£‡πâ‡∏≤‡∏á‡∏•‡∏≥‡∏î‡∏±‡∏ö Farey ‡∏î‡πâ‡∏ß‡∏¢‡∏Å‡∏≤‡∏£‡πÄ‡∏û‡∏¥‡πà‡∏°‡∏™‡∏°‡∏≤‡∏ä‡∏¥‡∏Å‡∏•‡∏á‡πÉ‡∏ô‡∏•‡∏≥‡∏î‡∏±‡∏ö‡πÑ‡∏õ‡πÄ‡∏£‡∏∑‡πà‡∏≠‡∏¢‡πÜ ‡πÇ‡∏î‡∏¢‡πÉ‡∏ä‡πâ‡∏Ç‡πâ‡∏≠‡∏°‡∏π‡∏•‡∏à‡∏≤‡∏Å‡∏™‡∏°‡∏≤‡∏ä‡∏¥‡∏Å‡∏ó‡∏µ‡πà‡∏°‡∏µ

‡∏≠‡∏≤‡∏®‡∏±‡∏¢‡∏™‡∏°‡∏ö‡∏±‡∏ï‡∏¥ 1.5 ‡πÅ‡∏•‡∏∞‡∏ó‡∏§‡∏©‡∏é‡∏µ‡∏ö‡∏ó‡∏ó‡∏µ‡πà 2.5 ‡∏Å‡∏≤‡∏£‡∏™‡∏£‡πâ‡∏≤‡∏á $F_n$ ‡∏î‡πâ‡∏ß‡∏¢‡∏ß‡∏¥‡∏ò‡∏µ‡∏ô‡∏µ‡πâ ‡πÄ‡∏û‡∏µ‡∏¢‡∏á‡πÅ‡∏ï‡πà‡∏ó‡∏£‡∏≤‡∏ö 2 ‡πÄ‡∏ó‡∏≠‡∏°‡πÅ‡∏£‡∏Å‡πÉ‡∏ô‡∏•‡∏≥‡∏î‡∏±‡∏ö‡∏Å‡πá‡∏™‡∏≤‡∏°‡∏≤‡∏£‡∏ñ‡∏´‡∏≤‡∏™‡∏°‡∏≤‡∏ä‡∏¥‡∏Å‡∏ó‡∏µ‡πà‡πÄ‡∏´‡∏•‡∏∑‡∏≠‡∏Ç‡∏≠‡∏á‡∏•‡∏≥‡∏î‡∏±‡∏ö‡πÑ‡∏î‡πâ

``` python
def farey(n):
    sequence = [f, g] = [Fraction(0), Fraction(1, n)]
    while g < 1:
        k = (n + f.denominator) // g.denominator
        a = k * g.numerator - f.numerator
        b = k * g.denominator - f.denominator
        f, g = g, Fraction(a, b)
        sequence += [g]
    return sequence
```


## ‡∏ß‡∏á‡∏Å‡∏•‡∏° Ford

**‡∏ô‡∏¥‡∏¢‡∏≤‡∏° 3** ‡∏™‡∏≥‡∏´‡∏£‡∏±‡∏ö‡πÄ‡∏®‡∏©‡∏™‡πà‡∏ß‡∏ô‡∏≠‡∏¢‡πà‡∏≤‡∏á‡∏ï‡πà‡∏≥ $p/q$ ‡∏ô‡∏¥‡∏¢‡∏≤‡∏°‡∏ß‡∏á‡∏Å‡∏•‡∏° Ford ‡∏î‡πâ‡∏ß‡∏¢‡∏™‡∏±‡∏ç‡∏•‡∏±‡∏Å‡∏©‡∏ì‡πå $C(p,q)$ ‡∏Ñ‡∏∑‡∏≠‡∏ß‡∏á‡∏Å‡∏•‡∏°‡∏ó‡∏µ‡πà‡∏°‡∏µ‡∏à‡∏∏‡∏î‡∏®‡∏π‡∏ô‡∏¢‡πå‡∏Å‡∏•‡∏≤‡∏á‡∏ó‡∏µ‡πà $\left( \frac{p}{q}, \frac1{2q^2} \right)$ ‡πÅ‡∏•‡∏∞‡∏°‡∏µ‡∏£‡∏±‡∏®‡∏°‡∏µ $\frac1{2q^2}$

{: .figure}
> ![](/images/math/ford-circles.png)
>
> ‡∏†‡∏≤‡∏û 1: ‡∏ß‡∏á‡∏Å‡∏•‡∏° Ford ‡∏ó‡∏µ‡πà‡∏™‡∏£‡πâ‡∏≤‡∏á‡∏à‡∏≤‡∏Å‡πÄ‡∏®‡∏©‡∏™‡πà‡∏ß‡∏ô‡∏Ç‡∏≠‡∏á‡∏•‡∏≥‡∏î‡∏±‡∏ö Farey ‡∏≠‡∏±‡∏ô‡∏î‡∏±‡∏ö 5

**‡∏ó‡∏§‡∏©‡∏é‡∏µ‡∏ö‡∏ó 3.1** ‡πÑ‡∏°‡πà‡∏°‡∏µ‡∏ß‡∏á‡∏Å‡∏•‡∏° Ford ‡∏Ñ‡∏π‡πà‡πÉ‡∏î‡πÄ‡∏•‡∏¢‡∏ó‡∏µ‡πà‡∏ã‡πâ‡∏≠‡∏ô‡∏Å‡∏±‡∏ô

**‡∏û‡∏¥‡∏™‡∏π‡∏à‡∏ô‡πå** ‡πÉ‡∏´‡πâ $R$ ‡πÅ‡∏ó‡∏ô‡∏£‡∏∞‡∏¢‡∏∞‡∏´‡πà‡∏≤‡∏á‡∏£‡∏∞‡∏´‡∏ß‡πà‡∏≤‡∏á‡∏à‡∏∏‡∏î‡∏®‡∏π‡∏ô‡∏¢‡πå‡∏Å‡∏•‡∏≤‡∏á‡∏Ç‡∏≠‡∏á $C(a,b), C(c,d)$ ‡πÅ‡∏•‡∏∞‡πÉ‡∏´‡πâ $r_{C(a,b)}, r_{C(c,d)}$ ‡πÅ‡∏ó‡∏ô‡∏£‡∏±‡∏®‡∏°‡∏µ‡∏Ç‡∏≠‡∏á‡∏ß‡∏á‡∏Å‡∏•‡∏°‡∏ó‡∏±‡πâ‡∏á‡∏™‡∏≠‡∏á‡∏ï‡∏≤‡∏°‡∏•‡∏≥‡∏î‡∏±‡∏ö  

$$
\begin{align}
R &= \sqrt{ \left( \frac{c}{d}-\frac{a}{b} \right)^2
          + \left( \frac1{2d^2} - \frac1{2b^2} \right)^2 } \\
    &= \frac1{2b^2d^2} \sqrt{4b^2d^2(bc-ad)^2 + (b^2-d^2)^2} \\
    &\ge \frac1{2b^2d^2} \sqrt{4b^2d^2 + (b^2-d^2)^2} \\
    &= \frac1{2b^2} + \frac1{2d^2} \\
    &= r_{C(a,b)} + r_{C(c,d)}
\end{align}
$$

‡πÄ‡∏û‡∏£‡∏≤‡∏∞‡∏£‡∏∞‡∏¢‡∏∞‡∏´‡πà‡∏≤‡∏á‡∏à‡∏≤‡∏Å‡∏®‡∏π‡∏ô‡∏¢‡πå‡∏Å‡∏•‡∏≤‡∏á‡∏Ç‡∏≠‡∏á‡∏™‡∏≠‡∏á‡∏ß‡∏á‡∏Å‡∏•‡∏° ‡∏°‡∏µ‡∏Ñ‡πà‡∏≤‡∏°‡∏≤‡∏Å‡∏Å‡∏ß‡πà‡∏≤‡∏£‡∏±‡∏®‡∏°‡∏µ‡∏£‡∏ß‡∏°‡∏Å‡∏±‡∏ô ‡∏î‡∏±‡∏á‡∏ô‡∏±‡πâ‡∏ô‡∏à‡∏∂‡∏á‡πÑ‡∏°‡πà‡∏°‡∏µ‡∏ß‡∏á‡∏Å‡∏•‡∏° Ford ‡∏Ñ‡∏π‡πà‡πÉ‡∏î‡∏ã‡πâ‡∏≠‡∏ô‡∏Å‡∏±‡∏ô

**‡∏ó‡∏§‡∏©‡∏é‡∏µ‡∏ö‡∏ó 3.2** ‡∏ß‡∏á‡∏Å‡∏•‡∏° $C(a,b), C(c,d)$ ‡∏ó‡∏µ‡πà‡∏™‡∏±‡∏°‡∏ú‡∏±‡∏™‡∏Å‡∏±‡∏ô‡πÅ‡∏•‡∏∞ $\frac{a}{b} < \frac{d}{c}$ ‡∏°‡∏µ‡∏à‡∏∏‡∏î‡∏™‡∏±‡∏°‡∏ú‡∏±‡∏™‡∏ó‡∏µ‡πà $\left( \frac{a}{b} + \frac1{b^2+d^2}\frac{d}{b}, \frac1{b^2+d^2} \right)$

**‡∏û‡∏¥‡∏™‡∏π‡∏à‡∏ô‡πå** ‡∏à‡∏≤‡∏Å‡∏™‡∏π‡∏ï‡∏£ $(x,y) = \left( \frac{mx_2+nx_1}{m+n}, \frac{my_2+ny_1}{m+n} \right)$

- ‡∏ß‡∏á‡∏Å‡∏•‡∏° $C(a,b)$ ‡∏°‡∏µ‡∏à‡∏∏‡∏î‡∏®‡∏π‡∏ô‡∏¢‡πå‡∏Å‡∏•‡∏≤‡∏á‡∏≠‡∏¢‡∏π‡πà‡∏ó‡∏µ‡πà $(x_1,y_1) = (\frac{a}{b}, \frac1{2b^2})$ ‡πÅ‡∏•‡∏∞ $m = r_1 = \frac1{2b^2}$
- ‡∏ß‡∏á‡∏Å‡∏•‡∏° $C(c,d)$ ‡∏°‡∏µ‡∏à‡∏∏‡∏î‡∏®‡∏π‡∏ô‡∏¢‡πå‡∏Å‡∏•‡∏≤‡∏á‡∏≠‡∏¢‡∏π‡πà‡∏ó‡∏µ‡πà $(x_2,y_2) = (\frac{c}{d}, \frac1{2d^2})$ ‡πÅ‡∏•‡∏∞ $n = r_2 = \frac1{2d^2}$

‡∏î‡∏±‡∏á‡∏ô‡∏±‡πâ‡∏ô ‡∏à‡∏∏‡∏î‡∏™‡∏±‡∏°‡∏ú‡∏±‡∏™‡∏Ç‡∏≠‡∏á‡∏ß‡∏á‡∏Å‡∏•‡∏° $C(a,b), C(c,d)$ ‡∏à‡∏∞‡∏≠‡∏¢‡∏π‡πà‡∏ó‡∏µ‡πà

$$
\begin{align}
(x,y) &= \left( \frac{r_1x_2+r_2x_1}{r_1+r_2}, \frac{r_1y_2+r_2y_1}{r_1+r_2} \right) \\
      &= \left( \frac{r_2}{r_1+r_2}\frac{a}{b} + \frac{r_1}{r_1+r_2}\frac{c}{d}, \frac{r_1r_2+r_2r_1}{r_1+r_2} \right) \\
      &= \left( \frac{r_1+r_2}{r_1+r_2}\frac{a}{b} + \frac{r_1}{r_1+r_2}(\frac{c}{d}-\frac{a}{b}), \frac{2r_1r_2}{r_1+r_2} \right) \\
      &= \left( \frac{a}{b} + \frac1{2b^2}\frac{2b^2d^2}{b^2+d^2}\frac{bc-ad}{bd}, 2\frac1{2b^2}\frac1{2d^2}\frac{2b^2d^2}{b^2+d^2} \right) \\
      &= \left( \frac{a}{b} + \frac1{b^2+d^2}\frac{d}{b}, \frac1{b^2+d^2} \right)
\end{align}
$$
